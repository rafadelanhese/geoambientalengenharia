Regras básicas de interação em cadastros


Padrões e Ícones:


Botão


Nome


Descrição

Incluir

Ao pressionar o botão uma nova tela irá abrir para a inclusão de um novo registro

Visualizar

Ao pressionar o botão uma janela irá abrir com informações pertinentes ao item mostrado na lista

Editar

Ao pressionar o botão a tela de cadastro irá abrir com os campos preenchidos para edição do item escolhido

Excluir

Ao pressionar o botão o item desejado será excluído do banco de dados


Formas de interação:


Como Incluir um Cadastro

Pressinando o botão +Incluir, a tela de cadastro irá abrir para preenchimentos das informações.


Como Visualizar um Cadastro

Clicando no botão de Visualizar uma janela irá se abrir com as informações do item desejado.  


Como Alterar um Cadastro

Para alterar um cadastro, basta clicar no botão Editar e a tela de cadastro irá abrir com as informações preenchidas para edição.

 

Como Excluir um Cadastro

Para excluir um cadastro, assim como para alterar, basta selecioná-lo e clicar no botão excluir. A eliminação é possível sempre que o cadastro não tenha qualquer relacionamento ativo , isto é, que não exista ligação desses itens com uma atividade ou pessoa, por exemplo.
