Procedimentos para instalação/configuração do Software


Atualmente o software está hospedado no Integrator que oferece o suporte o suporte necessário. Para a alteração de hospedagem e/ou plano a nova hospedagem deve oferecer suporte aos seguintes componentes:

  • Java
  • Vraptor
  • Hibernate
  • Tomcat
  • MySQL

Instalando e Configurando o Banco de Dados

    No servidor deverá ser instalado o banco de dados MySQL disponível.

Nome do banco: geoambie_geoambiental
Usuário: geoambie_admin

Senha: x8p0Em!A#MqJ

Atenção: o nome do banco, usuário e senha são padrões do sistema, qualquer outro tipo de configuração o sistema não irá funcionar.


Instalando e Configurando o Software

    Instalalar a versão do Tomcat 7 no servidor.

Instalar o JDK: jdk1.8.0_144

Instalar o banco de dados MySQL como descrito a pouco.

Utilizando algum programa de FTP, localizar no servidor a pasta webapp/tomcat..versão do tomcat/

Nesta pasta enviar o .war do sistema
